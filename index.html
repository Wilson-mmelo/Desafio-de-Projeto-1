Introdução ao HTML na Prática

(Voltar)

HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar páginas web.

Ela fornece a estrutura básica de um site, permitindo que você organize e apresente conteúdo como texto, imagens, vídeos e outros elementos em um navegador

Apesar de ser uma linguagem de marcação, e não uma linguagem de programação, o HTML é fundamental para o desenvolvimento web, pois define como o conteúdo é estruturado e apresentado.

O HTML utiliza tags (elementos) para definir a estrutura de uma página web, como parágrafos, títulos, listas, tabelas, etc.

As tags são palavras-chave envoltas em colchetes angulares (< e>). Por exemplo, < p > e < /p > marcam o início e o fim de um parágrafo.

As tags podem ter atributos, que fornecem informações adicionais sobre o elemento. Por exemplo, < img src="imagem.jpg" alt="Descrição da imagem" > define uma imagem e seus atributos (fonte e texto alternativo) src e alt são atributos.

O HTML permite dar significado (semântica) ao conteúdo da página, indicando o que cada parte representa. Por exemplo, usar < header > para o cabeçalho, < nav > para a navegação, < h1 > para o título principal, e assim por diante.

Tags e elementos:

< p > : Define um parágrafo.

< h1 > a < h6 > : Define títulos, com < h1 > sendo o título principal e < h6 > o menor.

< img > : Define uma imagem, com o atributo src especificando o caminho da imagem e o atributo alt fornecendo texto alternativo.

< a > : Define um link, com o atributo href especificando o endereço do link.

< ul > , < ol >, < li > : Define listas não ordenadas e ordenadas, com < li > representando cada item da lista.

< table > , < tr >, < td > : Define tabelas, com < tr > representando uma linha e < td > representando uma célula da tabela.

Trabalhando com Formulários em HTML

(voltar)

Em HTML, formulários são usados para coletar dados do usuário e enviá-los para um servidor web.

Eles são compostos por elementos como campos de entrada, rótulos, botões e caixas de seleção, todos contidos dentro de uma tag < form >.

Para trabalhar com formulários em HTML, é importante entender como estruturá-los, quais elementos utilizar e como enviar os dados coletados.

Estrutura básica de um formulário HTML:

Um formulário HTML é definido pela tag < form >, que contém todos os elementos do formulário.

Dentro da tag < form >, você pode incluir:

  • < labe > : Define um rótulo para um campo de entrada, descrevendo o que o usuário deve inserir.
  • < input > : Cria campos de entrada interativos, como campos de texto, senhas, e-mails, etc. O tipo de campo é definido pelo atributo type.
  • < textarea > : Permite que o usuário insira textos longos.
  • < select > : Cria menus suspensos.
  • < button > : Cria botões que podem ser usados para enviar o formulário ou realizar outras ações.

Atributos importantes de um formulário:

action: Especifica o URL para onde os dados do formulário serão enviados.

method: Define o método HTTP para enviar os dados. Os métodos mais comuns são GET (dados na URL) e POST (dados no corpo da requisição).

Estruturando seu HTML+Formatações

(Voltar)

A estrutura básica de um documento HTML consiste em quatro elementos principais: <!DOCTYPE html >, < html >, < head >, e < body >.

O < !DOCTYPE html > declara o tipo de documento como HTML5, informando ao navegador como interpretá-lo. O elemento < html > é o elemento raiz, contendo todo o conteúdo da página.

Dentro do < html >, temos o < head >, que contém informações sobre o documento, como título e metadados, e o < body >, que contém o conteúdo visível da página, como texto, imagens e outros elementos.

A formatação do texto em HTML pode ser feita através de tags como < b > (negrito) , < i > (itálico) , < strong > (ênfase) , < em > (ênfase) entre outras.

Além disso, elementos como < p > (parágrafo) , < h1 > a < h6 > (cabeçalhos) e outros elementos estruturais ajudam a organizar o conteúdo da página de forma lógica e hierárquica.

Outras tags úteis:

  • < a > : Define um hiperlink.
  • < img > : Insere uma imagem.
  • < ul > , < ol > , < li > : Define listas não ordenadas, ordenadas e itens de lista, respectivamente.
  • < div > : Define uma seção genérica na página.
  • < span > : Define uma parte inline de um texto.

Trabalhando com Mídias utilizando HTML

(Voltar)

O atributos de mídia são usados para modificar o comportamento ou adicionar informações extras aos elementos multimídia, como imagens, áudio e vídeo. Eles fornecem detalhes sobre como o conteúdo deve ser exibido ou reproduzido.

A tag < img > é usada para incorporar uma imagem em uma página HTML.

As imagens não são tecnicamente inseridas em uma página da web, mas sim vinculadas a páginas da web. Essa tag cria um espaço de retenção para a imagem referenciada.

A tag < img > tem dois atributos obrigatórios: src - Especifica o caminho para a imagem e alt - Especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida

Sempre especifique a largura e a altura de uma imagem, caso contrário a página pode piscar enquanto a imagem é carregada.

Para vincular uma imagem a outro documento, basta aninhar a tag < img > dentro de uma tag < a >

Atributos da tag < img >:/p>

width: "200", se utilizar sozinho vai escalar a imagem de forma dinâmica entre largura e altura.

height: "200", em conjunto com width vai forçar a largura e altura definido podendo distorcer a imagem.

title: quando passamos o mouse em cima mostra o titulo.

sizes: Especifica tamanhos de imagem para diferentes layouts de página.

srcset: Especifica uma lista de arquivos de imagem para usar em diferentes situações.

A tag < audio > é usada para incorporar conteúdo de som em um documento, como música ou outros fluxos de áudio, pode contém uma ou mais tags com diferentes fontes de áudio, logo o navegador escolherá a primeira fonte compatível.

A tag < audio > é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo, podendo conter uma ou mais tags com diferentes fontes de vídeo, então o navegador escolherá a primeira fonte compatível.

O texto entre as tags < video > e < /video >só será exibido em navegadores que não suportam o elemento < video >.

Existem três formatos de vídeo suportados em HTML: MP4, WebM e OGG.

Atributos tag < video >:

autoplay: começa o vídeo automáticamente ao carregar o navegador;

controls: mostra os botões de controle de volume, velocidade e tempo.

tag filha:

source:

atributo: src, ou seja caminho do arquivo de vídeo;

< track > Este elemento é usado para especificar legendas, arquivos de legenda ou outros arquivos contendo texto, que devem estar visíveis quando a mídia estiver sendo reproduzida.

As trilhas são formatadas no formato WebVTT (arquivo.vtt), então é necessário além do arquivo de áudio ou vídeo, mais o arquivo vtt que é a legenda.

O atributo kind com valor descriptions pode ser usado como uma narração do que acontece para acessibilidade, então o texto colocado servirá de narração pelos equipamentos adaptados.

< iframe >A tag especifica um quadro embutido, que é usado para incorporar outro documento no documento HTML atual.

Criando tabelas com HTML

(Voltar)

Tabelas são usadas para exibir dados em formato tabular, organizados em linhas e colunas, utilizando as tags < table >, < tr >, < td >e, opcionalmente, < th >.

A tag < table > define o início e o fim da tabela, <tr > representa uma linha, e < td > define uma célula de dados.

Opcionalmente, < th > pode ser usado para definir células de cabeçalho, comumente usadas para títulos de colunas.

Estrutura básica de uma tabela HTML:

  • < table > : Define a tabela em si.
  • < tr > : Define uma linha na tabela.
  • < td > : Define uma célula de dados dentro de uma linha (coluna).
  • < th > (opcional) : Define uma célula de cabeçalho dentro de uma linha (coluna).

Outras tags e atributos úteis:

  • < colgroup > e < col > : Permitem definir estilos para grupos de colunas
  • <thead >, < tbody >, < tfoot> : Permitem agrupar o cabeçalho, corpo e rodapé da tabela, respectivamente, para melhor organização e estilização.
  • < colspan > e < rowspan > : Permitem mesclar células horizontalmente e verticalmente, respectivamente.

Entendendo HTML Semântico

(Voltar)

HTML semântico é o uso de tags HTML que fornecem significado e estrutura ao conteúdo de uma página web, em vez de simplesmente definir sua aparência.

Ao invés de usar elementos genéricos como < div > e < span > para tudo, o HTML semântico emprega tags como < header >, < nav > , < article >, < aside >, < footer > e < section >, que descrevem a função de cada parte do conteúdo.

Isso torna o código mais legível para desenvolvedores, facilita a indexação pelos mecanismos de busca e melhora a acessibilidade para usuários com deficiência.

Embora o HTML5 tenha introduzido novas tags semânticas, o HTML semântico é suportado por todos os navegadores modernos e também é compatível com versões mais antigas.

Exemplos de tags semânticas:

  • < header > : Define o cabeçalho de uma página ou seção.
  • < main > : Define o conteúdo principal da página.
  • < nav > : Define uma seção de navegação.
  • < article > : Define um conteúdo independente e auto-contido, como um artigo de blog.
  • < aside > : Define conteúdo relacionado ao conteúdo principal, mas que pode ser considerado separado.
  • < footer > : Define o rodapé da página ou seção.
  • < section > : Define uma seção temática da página.
