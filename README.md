No Módulo 01, o desafio foi criar um site simples utilizando todas as tags HTML ensinadas nos primeiros passos como <h1> a <h6>, <p>, <strong>, listas, links, entre outras. 
Também foi solicitado fazermos uma pesquisar e aplicar novas tags como <font>, <del>, e <abbr>. 
O objetivo é fixar bem esses elementos, pois serão usados ao longo de todo o curso. 
É normal termos dúvidas nesse processo de aprendizagem — elas nos ajudarão no aprendizado no decorrer de todo o curso.
